Setup Instructions

1. Backend (port 8000)

  1. cd backend
  2. python -m venv venv && venv\Scripts\activate
  3. pip install -r requirements.txt
  4. Create .env: GEMINI_API_KEY=your_key
  5. uvicorn main_fixed_fixed:app --reload or python main_fixed_fixed.py

2. Frontend (port 3000)

  1. cd frontend
  2. npm install
  3. npm run dev

✅ Open http://localhost:3000 - Full UI ready!
